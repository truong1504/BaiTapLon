Code::Blocks 20.03 (32-bit)
https://sourceforge.net/projects/codeblocks/files/Binaries/20.03/Windows/32bit/
Download codeblocks-20.03-32bit-mingw-32bit-setup.exe

Mở file RobotMazePROJECT.cbp
Dịch project: Ctrl F11
Chạy project: Ctrl F10

Trong trường hợp không nhìn được toàn bộ giao diện:
1. Chuột phải vào thanh công cụ trên Console
2. Chọn Properties, chọn Layout
3. Chỉnh Width (120), Height (35) ở Screen Buffer Size và Window Size
4. Tắt (ESC) và chạy lại chương trình

Nếu có câu hỏi, email: khoi.n.le@vnu.edu.vn
